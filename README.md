# Sistema de Playlist

Projeto da disciplina de Estrutura de Dados - Fatec Rio Claro.

## Sobre

Backend de um aplicativo de músicas. Permite gerenciar uma biblioteca pessoal,
montar filas de reprodução por humor (com base no BPM da música) e consultar
o histórico de reproduções.

## Como executar

Basta rodar o menu na pasta do projeto:

```
python menu.py
```
